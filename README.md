- **[DUCKODUCKO 🦆](https://github.com/scottx611x/DUCKODUCKO)** — a whimsical top-down, tap-to-hop duck roguelike built in Godot. Play instantly → [duckoducko.scott-ouellette.com](https://duckoducko.scott-ouellette.com)
